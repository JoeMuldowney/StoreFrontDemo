# Sunny Side Café – Demo Website

Sunny Side Café is a modern, responsive demo website showcasing a fictional café serving breakfast and brunch. This project demonstrates best practices in performance, responsive design, accessibility, and SEO, suitable for freelance portfolios.

View the live demo: https://joemuldowneydemos-sunnysidecafe.com

# Features

1. **Performance**
    - Largest Contentful Paint (LCP) optimized for images.
    - Responsive images (srcset + sizes) in WebP format.
    - Minimal render-blocking resources; critical CSS and JS optimized.

2. **Accessibility**
    - Properly structured headings (H1 → H2 → H3…).
    - Buttons with accessible aria-labels.
    - Main landmark and semantic HTML for screen readers.
    - Social icons with descriptive alt text.

3. **SEO**
    - Meta description provided for search engines.
    - Robots.txt configured to allow crawling.
    - Sitemap.xml can be added to guide search engines.

4. **Responsive Design**
    - Mobile-first layout using Bootstrap 5.
    - Hero section scales with viewport height (vh).
     -Menu cards and images adjust for smaller screens.

5. **Images**
    - WebP format used for faster downloads.
    - Hero image uses fetchpriority="high" for LCP.
    - Menu images use srcset for multiple resolutions (400px, 800px).
    - SVG icons for social links for crisp rendering and small file size.
